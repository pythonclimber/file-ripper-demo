File-ripper is a lightweight library for parsing text files and unleashing the data inside. File-ripper lets you define the format of the file via its FileDefinition interface. File-ripper supports fixed-width, character delimited, and XML formats. Your data is returned in a simple format that can easily be converted to JSON for submitting to an API. Optionally, you can specify and object builder function to have your data returned in the data type of your choice.
